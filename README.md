# Set up APF

## Update
```
sudo apt update
sudo apt full-upgrade -y
# (Possibly: sudo apt install --no-install-recommends kali-linux-everything -y)
```

## Get repo
```
# As non-root
cd ~
rm -rf DA-SetupKali/
git clone https://github.com/stewri707/DA-SetupKali.git
```
